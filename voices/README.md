# Röster

Placera dina nerladdade Piper-röster här.

En röst består alltid av två filer. Till exempel:
- `sv_SE-alma-medium.onnx`
- `sv_SE-alma-medium.onnx.json`

Appen letar automatiskt igenom den här mappen och gör alla tillgängliga .onnx-filer valbara, förutsatt att motsvarande .json-fil också existerar.
